# 起因

这是整理手机清理出来的最近两年的吃饭图片，我就直接把他们放在这儿了。

主要是需要做的

1. 压缩图片
2. 列出文件引用地址

当然需要安装下 `brew install imagemagick` 作为图片压缩的工具。

这两个都可以利用已经写好的 Python 程序完成。
